# Vue user interface for a handcraft store (Artesanaria)

## Clone the Project

```
git clone https://github.com/Kikemn/vue-artesanaria-template.git
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
